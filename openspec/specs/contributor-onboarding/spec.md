## Requirements
### Requirement: Repository SHALL provide contributor onboarding guidance
The repository MUST include a contributor guide covering prerequisites, build steps, test entry points, and verification expectations for local changes.

#### Scenario: New contributor starts local setup
- **WHEN** a developer opens the repository for the first time
- **THEN** they can identify the required toolchain, the main validation commands, and known environment pitfalls without reverse-engineering maintainer workflow
